Online Retail Recommendation System

Project Overview
Python project that recommends products to customers based on past purchase history, similar to Amazon or Netflix recommendation systems.

Features
- Clean and preprocess dataset
- Build Customer-Product Matrix
- Recommend similar products using cosine similarity

Project Structure
online-retail-recommendation-system/
--data/ Dataset folder
 --Online Retail.xlsx
--notebooks/  Optional Jupyter notebooks
--src/
 ---main.py  Main Python script
 --README.md
--requirements.txt

How to Run
1. Install required libraries:  
```bash
pip install -r requirements.txt
2. Run main script:
python src/main.py
